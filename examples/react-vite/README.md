# ZINTENT + React (Vite)

Minimal example: React 18 + Vite, with CSS generated by the ZINTENT compiler from `index.html` and everything under `src/`.

## Prerequisites

- Go 1.22+
- Node.js 20+

From the **repository root** (`ZINTENT/`):

## One-time setup

```bash
cd examples/react-vite
npm install
```

## Generate CSS

```bash
# From repo root
go run ./compiler --input examples/react-vite/index.html -o examples/react-vite/public/zintent.css --preset core --content examples/react-vite/src
```

Or from this folder (adjust paths if your clone location differs):

```bash
cd examples/react-vite
npm run build:css
```

## Dev server

```bash
npm run dev
```

Vite serves `public/zintent.css` at `/zintent.css` (reload after `build:css` when you change classes).

## Production build

```bash
npm run build
```

`build` runs `build:css` then `vite build`.

## Notes

- Use `className="..."` in JSX; the compiler’s regex scanner picks up string literals.
- Prefer `--preset core` or `minimal` for smaller output.
- See `docs/MIGRATION_FROM_TAILWIND.md` if you come from Tailwind.
